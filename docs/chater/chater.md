# プロジェクト憲章: webアプリ「世界時計」の開発

webアプリ「世界時計」の開発

## スコープ
- webアプリ「世界時計」の開発
- 開発環境の構築

## 成果物
- webアプリ「世界時計」の開発資料（仕様書、設計書、テスト仕様書等）
- プログラムコード
- 開発環境の仕様
- copilotを活用したwebアプリ開発のフロー（私とcopilotのやり取りから、手順を抽出したもの）

- 初回リリース：1か月後
- その後、毎週更新版をリリース

- PO（プロダクトオーナー）：私

## 開発方針
- WebアプリはNode.jsとhttpモジュールを用いて実装
- WindowsアプリはWebアプリ完成後に着手
- 仕様書・設計書・その他資料はMarkdown形式で管理
- コードは`http`と`node.js`を中心に記述

## フォルダ構成（予定）
- docs/ : 仕様書・設計書等のMarkdown資料
- src/  : Webアプリのソースコード
- win/  : Windowsアプリのソースコード（Webアプリ完成後に作成）
